# Todo App (MERN Stack)

## 📌 Description
This is a simple Todo Application built using MERN Stack (MongoDB, Express, React, Node.js).  
Users can add and delete todos in real-time.

---

## 🚀 Features
- Add new todo
- Delete todo
- Fetch todos from database
- Real-time updates

---

## 🛠️ Technologies Used
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

---

## ⚙️ How to Run

### Backend
1. Go to backend folder
2. Run:
   npm install
   node server.js

### Frontend
1. Go to frontend folder
2. Run:
   npm install
   npm run dev

---

## 🌐 Live Demo
Frontend is deployed on Netlify.

---

## 📌 Note
Backend runs locally. For full functionality, run backend using:

cd backend  
npm install  
node server.js  

---

## 🌐 API Endpoints
- GET /api/todos → Get all todos
- POST /api/todos → Add todo
- DELETE /api/todos/:id → Delete todo
